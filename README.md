# Project 3

## How to run

    $ cd project3
    $ npm install
    $ node app

## How to use

### Presenter
+   Sign up from signup page
+   Login from top page
+   Click 'Presentatio DEMO'

### Listener
+   Click 'Show presentatn DEMO' from top page

## プレゼンテーションスライドの操作方法
+   'Right' => Next page
+   'Left' => Previous page
+   'Enter' => Progress page
+   '0' => Reset view count (Statistics がおかしくなったらこれを実行)

## Presenter のみの機能
+   手書き (ドラッグ)
+   Listener のページを追従させる (ただし、Presenter と違うページを見ている Listener のページは追従させない)

## Statistics の見方
+   Red => Presenter
+   Blue => Listener
